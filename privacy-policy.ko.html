PhotoMimic 개인정보처리방침

시행일자: 2026-07-09 최종 수정일: 2026-07-09

PhotoMimic("우리")은 사용자의 개인정보를 보호하기 위해 한국 「개인정보 보호법」(이하 "개보법")에 따라 다음과 같이 처리방침을 수립·공개합니다.

1. 수집하는 개인정보 항목 및 수집 방법

구분 항목 수집 방법 및 저장 위치
계정 (선택) Apple userIdentifier, 이름·이메일(가입 시 사용자가 제공에 동의한 경우) Sign in with Apple. 이용자 기기(Keychain)에만 저장되며 회사 서버로 전송되지 않음
서비스 이용 익명 설치 식별자(무작위 UUID), 무료 분석 사용 횟수, 최초·최종 사용 시각 사진 분석 요청 시 자동 수집, 회사 서버에 저장
결제 App Store 구독 영수증(JWS 토큰) Pro 구독 확인 시 요청에 포함되어 검증에만 사용. 서버에 저장하지 않음(무상태 검증)
콘텐츠 분석 요청한 참고 사진 1장(축소되고 EXIF가 제거된 JPEG) 및 촬영 수치(초점거리, 중력 벡터, 얼굴 기울기 각도) 분석 요청 시 전송. 분석 완료 즉시 폐기, 저장하지 않음
문의 이메일 주소(회신용, 선택), 문의 내용, 앱 버전·기기 모델·OS 버전·익명 ID 해시 사용자가 문의 메일을 발송할 때

수집하지 않는 것: 위치정보(GPS 등 — 사진의 위치 EXIF는 전송 전 기기에서 제거됨), 연락처, 사진첩 전체 인덱스, 광고 식별자(IDFA), 주민등록번호. 크래시 리포팅·행태 분석 SDK를 사용하지 않습니다.

2. 개인정보의 처리 목적

  • 카메라 보정 가이드 제공 (참고 사진 분석)
  • 무료 이용 한도 관리, 구독 자격(entitlement) 확인 및 부정 사용 방지
  • 사용자 문의 응답

3. 보유 및 이용 기간

항목 기간
Apple userIdentifier·이름·이메일 이용자 기기에만 저장. 로그아웃 또는 계정 삭제 시 기기에서 즉시 삭제
익명 설치 식별자·무료 사용량 기록 부정 사용 방지 목적으로 보유. 계정 삭제(또는 이메일 요청) 시 식별자를 복원 불가능한 일방향 해시로 즉시 변환(분리 보관) — 무료 한도 부정 재설정 방지 목적으로만 사용
참고 사진 본문 분석 직후 즉시 서버 메모리에서 폐기 (저장하지 않음)
분석 결과 메타데이터 이용자 기기에만 저장 (회사 서버에 없음)
서버 접속 로그 (요청 경로·상태 코드·처리 시간 — 개인 식별자 미포함) 90일
사용자 문의 메일 응답 후 1년

4. 제3자 제공

PhotoMimic은 사용자의 개인정보를 제3자에게 판매·제공하지 않습니다. 법령에 의한 요구가 있는 경우에 한해 관련 기관에 제공할 수 있습니다.

5. 처리위탁

수탁자 위탁 업무 보유·이용 기간
Apple Inc. (US) 로그인(Sign in with Apple) / 결제(App Store IAP) Apple 정책
분석 API 운영 (사진 분석·무료 사용량 관리) 사용자 삭제 요청 시까지

6. 개인정보의 국외 이전

수탁자 이전 국가 이전 항목 이전 목적
Apple 미국·아일랜드 등 userIdentifier, 결제 영수증 로그인·결제

이전 거부 시 Apple 로그인·구독 기능 이용이 제한될 수 있으나 앱의 핵심 기능은 정상 이용 가능합니다.

7. 개인정보의 파기 절차 및 방법

  • 파기 시점: 보유 기간 만료 또는 사용자가 삭제를 요청한 때
  • 파기 절차: 삭제 요청 접수 후 지체 없이 파기 (앱 내 계정 삭제는 즉시 서버에 삭제 요청을 전송)
  • 파기 방법: 전자 파일은 복구 불가능한 방법으로 영구 삭제. 무료 사용량 기록의 식별자는 복원 불가능한 일방향 해시로 변환하여 원 식별자를 파기
